# Flashcards App

Pequeña aplicación react para estudiar vocabulario mediante "flashcards".

[Flashcards App](https://flashcards-tawny.vercel.app/)

## Tecnologías usadas.

- **React**
- **Typescript**
- **Vite**
- **Tailwind**
- **SWR** *(Se simula una api con ficheros .json en el directorio public)*
- **React Router Dom**
